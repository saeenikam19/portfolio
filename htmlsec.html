College Website Project

College Online Voting System

Description:
A secure web-based voting system developed for college elections such as CR/GR or council elections. The system allows authenticated students to cast their votes online while ensuring transparency, security, and prevention of duplicate voting.

Key Features

  • Secure student login using college credentials
  • Role-based access for Admin and Students
  • One-student-one-vote validation
  • Candidate nomination and approval
  • Real-time vote counting and result display
  • Election control (start/end) by admin

Tech Stack

  • Frontend: HTML, CSS, JavaScript, Bootstrap
  • Backend: Python (Flask/Django) / Java / PHP
  • Database: MySQL
  • Tools: VS Code, Git

What I Learned

  • User authentication and authorization
  • Database design and relationships
  • Backend logic and CRUD operations
  • Building real-world problem-solving applications

Outcome

The project successfully demonstrated a transparent and efficient digital voting process, reducing manual effort and ensuring fair elections within the college environment.
