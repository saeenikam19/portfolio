This is College Website Project

  • This is a simple college website built using HTML. It includes the following sections:
  • Header: Displays the college name, slogan ("Excellence in Education"), and a navigation menu.
  • Home: Welcome message with a full-width image of the college.
  • About Us: Brief info about the college history and mission.
  • Courses Offered: Lists programs like B.Sc in Computer Science, BBA, and Engineering.
  • Admissions: A form for prospective students to submit their name, email, and course of interest.
  • Faculty: Highlights experienced teaching staff.
  • Contact Us: Provides an email (note: misspelled as "collage.com") and phone number.
  • Extras: Link to more college images on Pixabay.
